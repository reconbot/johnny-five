Galileo-IO, which is Johnny-Five's IO-Plugin for the Intel Galileo and Edison platforms will now default to I2C bus 1 when running on the Edison Mini board.

This change will not affect code running on the Edison Arduino board or either generation of Galileo boards. The decision to default to bus 1 is the result of learning that SparkFun Edison Blocks with support for I2C are connecting on that bus, and not bus 6 (which the Edison Arduino Board connects on). Here are the physical pin outs for connecting on the Mini board:

Connection to bus 1:

I2C-1-SDAI2C-1-SCL
J17-8J18-6

Please report any issues here.
